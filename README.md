# Passagens Fixas do Zero

Landing page do curso Passagens Fixas do Zero — milhas do zero.

Single-file HTML, sem dependências externas exceto Google Fonts.

## Estrutura

- `index.html` — landing principal
- `assets/` — fotos, prints e depoimentos

## Assets necessários

Dentro de `assets/`:

- `davi-hero.png` — foto PNG sem fundo do Davi (hero)
- `davi-bio.jpg` — foto do Davi (seção quem é)
- `compare-antes.jpg` — print da passagem cara (comparativo)
- `compare-depois.jpg` — print da passagem com tabela fixa (comparativo)
- `depoimento-01.webp` até `depoimento-06.webp` — prints de WhatsApp dos depoimentos
